# Citation

GPU_REML is research software. A formal manuscript or software citation has not
yet been assigned.

For now, cite the repository URL and commit hash used in your analysis. Example:

```text
GPU_REML developers. GPU_REML: GPU-accelerated REML estimation for genomic
variance components. Git repository, commit <commit-hash>, <year>.
```

To record the exact version used:

```bash
git rev-parse HEAD
```
